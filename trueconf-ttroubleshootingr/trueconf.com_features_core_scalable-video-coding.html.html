Scalable SVC Infrastructure

Use TrueConf video conferencing solutions even when working on weak communication channels! Thanks to the support of scalable video coding (SVC) technology, each participant of the conference receives optimal video and audio quality.

Download for free

Automatic Resolution Selection

The video conferencing server automatically selects the resolution of the transmitted frame according to the conference layout, the channel bandwidth, and the capabilities of each participant's device.*

Network Load Optimization

The use of SVC technology can significantly reduce the load on the network, both on the server side and client applications.

* The analysis takes into account the size of each user's video window in the layout, the quality of the connection, and the performance of the device.

How does the SVC technology work?

The endpoint device divides the video into layers (SVC streams) and sends them to the video conferencing server.

The video conferencing server selects and sends the optimal set of layers for each client device based on network capabilities and the requested layout.

The client application on the device generates a layout from the received streams on the fly.
