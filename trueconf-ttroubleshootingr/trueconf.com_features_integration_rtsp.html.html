Software for Video Surveillance via RTSP

What Is The RTSP Protocol?

RTSP (Real Time Streaming Protocol) is a protocol which provides a simple set of basic commands to control the video stream. In order to connect to the camera via RTSP, you should send a special request to TrueConf Server. This request must contain a command (e.g. PLAY - to start broadcasting), IP address, and the path to the camera.

Connecting RTSP sources and IP cameras in video conference

The RTSP protocol allows any TrueConf user to connect to IP cameras and other sources of media content that support broadcasting through this protocol and monitor remote objects. Users may also connect cameras and broadcast the image from them during a video conference. Learn more →

Thanks to TrueConf software for video surveillance, users can not only connect to IP cameras, but also broadcast video conferences on RTSP players and media servers. Read more about TrueConf RTSP streaming →

The Benefits of Software for Video Surveillance

  • Having set the IP camera in an office or at the industrial shop floor, you can connect to it and control your company's premises or employee performance.
  • You can monitor remote objects 24/7 using the software for video surveillance. For example, if you go on vacation and do not want to leave the apartment unattended, just set one or more IP cameras there. By calling one of these cameras from your TrueConf client application, you can connect to your apartment at any time and get the real-time picture of what is happening there.
  • With TrueConf, you can record any video conference. If you have faced a problem during the surveillance process and managed to record it, the given record will be conclusive evidence in solving this issue.
